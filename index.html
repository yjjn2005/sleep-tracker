🔥 Firebase 실시간 동기화
멀티디바이스 실시간 동기화를 위해 Firebase Firestore를 사용합니다.

1단계 — console.firebase.google.com 접속
2단계 — 새 프로젝트 생성 → Firestore Database 활성화 (테스트 모드)
3단계 — 프로젝트 설정 → 앱 추가 → 웹앱 → Config 복사
4단계 — 아래에 JSON 형식으로 붙여넣기 후 연결
📱 GitHub Pages 배포 방법
1 — github.com 에서 새 퍼블릭 레포지토리 생성 (예: sleep-tracker)
2 — 이 index.html 파일을 레포에 업로드 (드래그 앤 드롭)
3 — Settings → Pages → Source: Deploy from branch → main
4 — username.github.io/sleep-tracker 로 접속
5 — iPhone, iPad, PC 등 모든 기기에서 같은 URL + 같은 사용자 키 입력
